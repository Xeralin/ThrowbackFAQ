Antivirus

Common antivirus issues and how to resolve them, including false-positive detections.

AV
My antivirus is blocking the game. What should I do?

Some antivirus programs flag game files as false positives. The fix is to add your game folder as an exclusion. For Windows Defender:

  1. Search for Virus & Threat Protection in the Windows start menu.
  2. Click Manage settings under "Virus & Threat Protection Settings."
  3. Scroll down to Exclusions and click "Add or remove exclusions."
  4. Click Add an exclusion, select Folder, and choose your game folder.
  5. Restart your PC and try launching the game again.

If the issue persists, make sure the exclusion is still in place, then run the downloader again to verify and re-download any removed files.

My antivirus deleted a game file. How do I get it back?
  1. Add your game folder as an exclusion in your antivirus settings.
  2. Run the downloader again to verify and re-download any missing files.
My browser is blocking the download. What do I do?

Some browsers block downloads that contain executable files. To get around this:

  • Chrome: Click the arrow next to the blocked download and select "Keep."
  • Edge: Click the three dots next to the blocked item and select "Keep."
  • Firefox: Click the download in the toolbar and select "Allow."

If that doesn't work, temporarily disable your browser's enhanced security or download protection, download the file, then re-enable it afterwards.

Only download files from the official Operation Throwback Discord server.

I use a third-party antivirus. Does the same apply?

Yes. The process is essentially the same for all antivirus software. You need to add your game folder as an exclusion. The exact steps vary by product, but look for an Exclusions, Exceptions, or Whitelist section in your antivirus settings.

If you're unsure how to do it for your specific antivirus, search for "[your antivirus name] add folder exclusion" for instructions on the internet.

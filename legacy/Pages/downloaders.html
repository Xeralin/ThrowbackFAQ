Frequently Asked Questions
Are the downloaders trojans or malware?

No. Your antivirus may flag them because they're not signed by a verified developer, obtaining a code-signing certificate requires a significant financial investment. If you have concerns, the source code for each downloader is available on GitHub for review.

Why do the downloaders need my Steam login?

Your credentials are required to access Steam's depot servers, which is where the old game files are stored. All downloaders use DepotDownloader, an open-source tool. Your credentials are sent directly to Steam, we have no way to store or intercept them.

Some downloaders will save your credentials to a local file in the resource folder, so you do not have to re-enter them each time you download a version. This file is stored only on your machine and is never transmitted anywhere.

What does "verify" mean and how do I do it?

Verifying checks for missing or corrupted files and re-downloads anything that needs fixing. On older downloaders, simply run the downloader again for the same season without deleting your existing files, it will handle the rest automatically. If you are using the JVAV downloader, a dedicated "Verify" option is available for this purpose.

How do I replace the crack files?

For newer crack:

  1. Download the latest crack ZIP from the official GitHub repository.
  2. Extract the ZIP. It contains crack files compatible with all seasons.
  3. Copy the contents into your season folder and replace any existing files when prompted.

For the old cracks:

  1. Download the Plazas ZIP from the Git Repo. Save it to a folder that has an antivirus exclusion.
  2. Extract the ZIP. It contains three subfolders for different season ranges:
    • Folder 1: Year 1 Season 1 — Year 6 Season 2
    • Folder 2: Year 6 Season 3 only
    • Folder 3: Year 6 Season 4 and later
  3. Open the correct folder for your game version, select all files, and copy them into your season folder. Replace existing files when prompted.

Using the wrong crack folder will prevent the game from launching. Double-check your season before replacing files.

Can I pause and resume the download?

Yes. Simply close the downloader at any point. When you reopen it and select the same season, it will verify existing files and continue from where it left off.

I'm getting errors while downloading. What should I do?

Most errors during download don't affect the final result and can be ignored. For specific errors:

  • Encountered error downloading chunk XXXXXX: This can be safely ignored. It just means the Steam servers were temporarily unreachable for that chunk. The downloader will automatically retry.
  • Depot XXXXX is not available: You don't own Siege on Steam, or you need to use the SKU RUS option.
  • Failed to allocate file: Free up storage space or move the downloader to a different drive.
  • Another process is using XXX: Close any program that might be interfering, such as your antivirus or another game instance.
How do I install the game to a different drive?
  • JVAV's / SlejmUr's downloader: Move the downloader to the desired location in File Explorer.
  • AKrisz's downloader: Change the download path in the downloader's settings.

Remember to update your antivirus exclusion to point to the new folder.

How do I delete a game version?

Delete the season folder from File Explorer. If the game is running in the background, close it via Task Manager first.

Do I need the current version of Siege installed?

No. Each downloaded version is completely independent and runs on its own, like a separate game.

My download is stuck at a certain percentage, is it broken?

Not necessarily. The progress bar only updates when an individual file finishes downloading, and some files are very large. Open Task Manager and check whether the downloader or .Net Host is using network bandwidth. If so, the download is still active.

What is SKU RUS and do I need it?

Some Steam accounts from post-USSR countries require a different regional version of the game. If your account is from one of the affected regions (shown in the image below), you need to select the SKU RUS option when downloading.

The SKU RUS version is in Russian by default. To change the language, download the localization.lang file from the #Guides-and-FAQ channel on Discord and place it in your game folder.

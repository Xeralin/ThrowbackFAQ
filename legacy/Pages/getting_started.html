Setup Guide

Getting Started

A step-by-step guide to setting up and downloading an older version of Rainbow Six Siege.

SETUP
// STEAM REQUIRED This guide only works if you own Rainbow Six: Siege on Steam. Ubisoft Connect and Epic Games accounts are not supported.
Step 1 — Prepare a Game Folder

Create a dedicated folder for the game on whichever drive you want to store it. Keep the name simple — no spaces or special characters. We recommend naming it OldR6.

// WARNING Do not place the game folder inside OneDrive or any other cloud storage service. This will cause issues.

Add an Antivirus Exclusion

Before downloading the game, add your game folder as an exclusion in your antivirus. This prevents the antivirus from interfering with game files. For Windows Defender:

  1. Search for Virus & Threat Protection in the Windows start menu.
  2. Click Manage settings under "Virus & Threat Protection Settings."
  3. Scroll down to Exclusions and click "Add or remove exclusions."
  4. Click Add an exclusion, select Folder, and choose your game folder.
Step 2 — Choose a Downloader

Several downloaders are available. Visit the Downloaders page for download links and detailed instructions.

JVAV's Downloader RECOMMENDED

  • Terminal-based
  • Batch/PowerShell-based — the most up-to-date option
  • Actively maintained

Legacy Downloaders

The following downloaders are no longer maintained. They may still work but are not recommended for new users.

  • AKrisz's Downloader — GUI-based, easier for those who prefer not to use a terminal
  • SlejmUr's Downloader — terminal-based, the original; many YouTube tutorials available
// NOTE Whichever downloader you choose, make sure to download it into your game folder before running it.
Step 3 — Download the Game
Using JVAV's Downloader (Recommended)
  1. Download the .bat file from the latest release and place it inside your game folder.
  2. Run the .bat file, it will automatically download everything it needs.
  3. Select Game Downloader from the main menu.
  4. Choose the year, then the season you want to download.
  5. Enter your Steam legacy username and password.
  6. Wait for the download to complete.
  7. Navigate to your game folder → downloads → [your selected season].
  8. Run launchr6.bat to start the game.

If the game gets stuck on the loading screen, download the resource folder from the Discord and place it in your game folder.

Using SlejmUr's Downloader (Legacy)
  1. Run the .exe or .bat file from inside your game folder.
  2. Follow the on-screen prompts and wait for countdowns to finish.
  3. Select Game Downloader when asked.
  4. Choose the season you want to download.
  5. Enter your Steam legacy username and password.
  6. Wait for the download to complete — "Download complete" will appear when done.
  7. Navigate to your game folder → downloads folder → season folder.
  8. Look for one of the following files and run it:
    • RainbowSix.bat
    • RainbowGame.exe
    • RainbowSix.exe

If you can't see file extensions, go to View → Show File name extensions. If multiple files are listed above, run the one highest in the list.

Using AKrisz's Downloader (Legacy)
  1. Run the setup file and follow the installation steps.
  2. Launch the downloader.
  3. Open Settings (bottom-left corner) and set the download folder to your game folder.
  4. Return to the home page and select the season you want.
  5. Click Download and wait for it to finish.
  6. Start the game using the Play button.
Need Help?

If you run into issues, check the Common Errors page or visit the How To Get Help page for guidance on reporting problems to staff.

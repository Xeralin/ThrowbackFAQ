Setup Guide

Multiplayer

How to set up and play with others using Radmin VPN.

MP
// NOTE This guide assumes you have already downloaded and set up the game. If not, start with the Getting Started guide.
Basic Setup

To play with others, everyone needs to be connected to the same Radmin VPN network. Follow these steps:

  1. Download and install Radmin VPN.
  2. Create a network in Radmin VPN or join one that your friends are already in.
  3. Make sure all players are connected to the same network.
  4. Start the game and create a Local Custom Game.
  5. Other players can join by selecting Join Local from the main menu.
Frequently Asked Questions
Does it matter who hosts the game?

Yes. The player with the most stable internet connection should host. If you're experiencing lag, try switching hosts.

I can't find the hosted game, what should I check?
  1. Check your game version: Both players must be on the same build. Enable "Show Metrics" in the game settings to see the version number.
  2. Check your Radmin VPN network: Make sure both players are connected to the same network.
  3. Check your firewall: Make sure the old Rainbow Six version is allowed through your firewall for both private and public networks.
  4. Restart: Try restarting both the game and Radmin VPN.
Can I use a different VPN instead of Radmin?

Yes, alternatives like Hamachi work. However, Radmin VPN is recommended because it supports more simultaneous connections per network. If you use a different VPN, we will not provide any support for any issues you may encounter.

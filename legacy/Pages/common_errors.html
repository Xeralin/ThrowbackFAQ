Common Errors

Solutions to the most frequently encountered game issues and errors.

ERR
I don't own Rainbow Six: Siege on Steam. Can I use my Ubisoft or Epic Games account?

No. The downloaders use Steam's depot service to download old game versions from Steam's servers. This requires a valid Steam account with a registered license for Rainbow Six: Siege. Ubisoft Connect and Epic Games accounts cannot be used to authenticate with Steam's servers.

Rainbow Six: Siege is free on Steam, so you can add it to your library at no cost. Once it's in your library the downloaders will work.

Can I use controller / Why is my game stuck on controller input?

Controllers are supported out of the box. Simply plug in your controller and it should be detected automatically. Note that PS5 (DualSense) controllers require additional third-party software (such as DS4Windows) to function properly.

If your game is stuck on controller input, try one of the following: disconnect your controller if its still connected and restart the game.

How do I fix the MSVCRXXX.dll error?

This error means your system is missing a required Microsoft Visual C++ Redistributable package. To fix it:

  1. Visit this page on GitHub.
  2. Download the ZIP file (not the source code).
  3. Extract the contents and run install_all.bat as administrator.
  4. Restart your PC and try launching the game again.

If the error persists, make sure Windows is fully up to date, then repeat the steps above.

How do I fix the uplay_rx_loader64.dll error?

This error usually occurs when your antivirus has blocked or removed a required file.

  1. Add your game folder as an exclusion in your antivirus settings.
  2. Re-apply the crack files from the Operation Throwback Discord server.
How do I fix missing or corrupt DLL files?

If you get an error mentioning any of the following files, the fix is the same for all of them:

  • amd_ags_x64.dll
  • gfsdk_ssao_d3d11.win64.dll
  • vivosxdk_x64.dll
  • bink2w64.dll
  1. Delete the specified .dll file from your game folder.
  2. Run the downloader again to verify and restore missing files.
My old Siege version opens the current version or Ubisoft Connect instead.
  1. Check your antivirus: Some crack files may be flagged and removed. Make sure your game folder has an exclusion set up.
  2. Re-apply the crack: Download the correct crack from the Operation Throwback Discord server.
  3. Replace files: Copy the contents of the crack folder into your game directory, replacing existing files.

Once done, try launching the game again.

My game is stuck on "Preparing Content", how do I fix it?
  1. Restart the game: Close it completely via Task Manager and try again.
  2. Verify your files: Run the downloader and let it check for missing or corrupted files.
  3. Check your antivirus: If the issue persists, your antivirus may have removed files. Exclude your game folder and re-apply the crack if needed.

This issue usually resolves itself after a restart. Verifying files is a reliable second step.

Why is my .exe / launch file missing?

If the executable is missing, it's usually caused by your antivirus removing it. To fix it:

  1. Add your game folder as an exclusion in your antivirus settings.
  2. Run the downloader again to verify and re-download any missing files.

If files keep going missing after verifying, double-check that your antivirus exclusion is set correctly before running the downloader again.

If verifying doesn't help, you may need SKU RUS. Visit the Downloaders page for more details.

My game crashes at the start screen after pressing the button.

If you're on an older crack, this is usually caused by a duplicate gamename in the .ini files. To fix it:

  1. Open codex.ini in your game folder.
  2. Find the GameName field and change it to any unique name.
  3. If you have multiple seasons installed, make sure each one has a different name.

This fix does not apply to the current Throwbackloader crack.

How do I change my in-game name or username?

Your username is stored in a config file in your game folder.

  • For the current Throwbackloader crack, open ThrowbackLoader.toml and edit the username field. Limit of 16 characters.
  • For older cracks, open codex.ini and edit the UserName and GameName fields.

Make sure to save the file before launching the game.

My game is in Russian. How do I switch to English?

If you installed the SKU RUS version and want to change the language:

  1. Download the localization.lang file from the #guides-and-faqs channel on the Operation Throwback Discord server.
  2. Move it to your game folder, replacing the existing file.
  3. Launch the game. It should now be in English.

If you're unsure whether you need SKU RUS, refer to the Downloaders page.

Why do I get a "User profile loading failed" error?

This error is expected and does not affect gameplay. Simply click OK and the game will continue as normal.

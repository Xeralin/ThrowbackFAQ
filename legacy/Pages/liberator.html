Tools & Mods

Liberator

Unlock all cosmetics and play additional game modes in Year 1 Season 1 through Year 4 Season 4.

LIB
What is Liberator?

Liberator is a tool for old Rainbow Six Siege that lets you unlock all cosmetics and access game modes that are otherwise unavailable, including Terrorist Hunt, playable solo or with others.

// IMPORTANT Liberator is only available through the official Operation Throwback Discord server. If you find it elsewhere, do not use it. We cannot verify its safety.
How to Use Liberator

Unlocking All Cosmetics

  1. Download Liberator from the Operation Throwback Discord server.
  2. Move the file into your game folder so Windows Defender won't block it.
  3. Run Liberator.exe.
  4. Launch the game and all cosmetics should be available.

Playing Custom Game Modes

  1. Run Liberator.exe first, before launching the game.
  2. Create a local custom game.
  3. Double-click the game mode you want to play in the Liberator menu. It will display text at the bottom of the Liberator window if it worked.
  4. If you want to play Terrorist Hunt or the Outbreak event, make sure you are on the blue team, then start the game.
Frequently Asked Questions
Windows Defender is blocking Liberator.exe, what do I do?

Move Liberator.exe directly into your game folder, which should already have an antivirus exclusion set up. If your browser blocks the download, temporarily disable browser protection to download it, then re-enable it afterward.

Only download Liberator from the official Operation Throwback Discord server.

Liberator is crashing the game, how do I fix it?

Make sure to launch Liberator before starting the game. If the game is still crashing, verify that it's fully closed, check Task Manager and end any remaining Rainbow Six Siege processes before trying again.

It says my build is unsupported, what does that mean?

Liberator only supports specific game builds. To check which builds are supported, open the About tab in the Liberator menu.

The game crashes when playing Terrorist Hunt or the Outbreak event with other players.

The order of operations matters here. Create the local custom game first and have other players join before selecting the game mode. Make sure everyone is on the blue team before starting.

Is there a Liberator for newer game versions?

There is no official Liberator for newer game versions. The closest alternative is the LiberatorV2 tool, which supports Shadow Legacy only. It is not affiliated with or supported by the Operation Throwback team. If you want to aquire it, you can find it in the Modding Tools section.
